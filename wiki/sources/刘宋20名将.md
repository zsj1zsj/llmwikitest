覆盖刘宋立国前后至宋明帝时期的主要将领，以结局为线索。

| **刘敬宣** | 被部下杀害 | 刘牢之之子，流亡南燕、复归东晋，支持刘裕，终遭部下杀害 |

| **傅弘之** | 力战被俘遭杀 | 北伐后秦七路进军之一，赫连勃勃青泥大败晋军，傅弘之力战被俘后遭杀 |
| **毛德祖** | 城破被俘不降 | 守司州抵抗北魏，城破被俘，至死不降 |
| **檀道济** | 被矫诏诛杀 | 见[[wiki/entities/檀道济|檀道济]]条目 |
| **沈庆之** | 被侄子闷杀 | 见[[wiki/entities/沈庆之|沈庆之]]条目 |

| **臧质** | 叛乱，湖中被杀 | 见[[wiki/entities/臧质|臧质]]条目 |

| **鲁爽** | 醉酒提前起兵被杀 | 见[[wiki/entities/鲁爽|鲁爽]]条目 |

| **蒯恩** | 被胡夏俘，客死异乡 | 随刘裕百余战，多次负重伤；迎桂阳公刘义真时被胡夏俘虏，卒于胡夏境内 |
| **沈林子** | 母丧悲郁病死 | 刘裕北伐水军将领，战功卓著，母亲去世后郁郁寡欢，很快病逝 |
| **吴喜** | 被宋明帝赐死 | 平三吴、平荆州立大功，宋明帝病危前因猜忌将其赐死 |

## 结局规律

- **死于内斗/猜忌**（11/20）：檀道济、沈庆之、谢晦、王镇恶、沈田子、柳元景、臧质、鲁爽、诸葛长民、刘毅、吴喜——刘宋功臣几乎无一善终
- **死于外敌**（5/20）：朱龄石、傅弘之、朱超石、何无忌、毛德祖
- **其他**（4/20）：刘敬宣（部下所杀）、蒯恩（客死胡地）、沈林子（病死）、沈攸之（叛乱自杀）
