# 魏晋南北朝基本史料目录

原文：[[raw/articles/魏晋南北朝基本史料目录]]

学术性书目，分七类，均为中华书局等权威点校本。

---

## 一、正史类（14种）

| 书名 | 撰者 | 特点 |
|------|------|------|
| 《后汉书》| 南朝宋·范晔 | 下限220年，含东汉末三国 |
| 《三国志》| 晋·陈寿 | 以魏为正统，过于简略；裴松之注丰富内容 |
| 《晋书》| 唐·房玄龄等 | 记载粗略，迷信成分多 |
| **《宋书》** | 梁·沈约 | **最为详尽**，各志多上溯制度源流 |
| 《南齐书》| 梁·萧子显 | 撰于王朝灭亡不久，史料价值高 |
| 《梁书》| 唐·姚思廉 | 全面叙述梁朝史事 |
| 《陈书》| 唐·姚思廉 | 正史中字数最少 |
| 《魏书》| 北齐·魏收 | 道武帝至东魏孝静帝 |
| 《北齐书》| 唐·李百药 | 高欢起兵至北齐灭亡约八十年 |
| 《周书》| 唐·令狐德棻等 | 西魏、北周史事 |
| 《隋书》| 唐·魏徵等 | 十志记载整个南北朝典章制度；经籍志尤重要 |
| 《南史》| 唐·李延寿 | 以家族为单位整合南朝各史 |
| 《北史》| 唐·李延寿 | 汇集正史，打破朝代断限 |
| 《新唐书·宰相世系表》| 宋·欧阳修 | 门阀家族世系，多追溯至南北朝 |

---

## 二、别史类（精选）

- 《十六国春秋辑补》（北魏·崔鸿）— 各国国书，"不附正朔，自相君长"
- 《资治通鉴》（宋·司马光，元·胡三省音注）— 上起前403年，下迄959年
- 《通鉴纪事本末》（宋·袁枢）— 以事为主，每事详书始末

---

## 三、地理类（精选）

- 《水经注疏》（北魏·郦道元）— 自然人文地理代表作
- 《洛阳伽蓝记》（北魏·杨衒之）— 北魏迁都邺城后追记洛阳佛寺之盛
- 《建康实录》（唐·许嵩）— 以建康为中心的六朝历史，含城池宫苑
- 《读史方舆纪要》（清·顾祖禹）— 考订古今郡县，叙山川险要

---

## 四、笔记小说类（精选）

- **《世说新语笺疏》**（南朝宋·刘义庆，余嘉锡笺疏）— 后汉末至东晋名士逸话
- 《颜氏家训集解》（北齐·颜之推）— 教育子孙，反映南北朝后期社会风俗
- 《异苑》（南朝宋·刘敬叔）— 先秦至刘宋怪异之事

---

## 五、百科/农工医术类（精选）

- 《齐民要术》（北魏·贾思勰）— 6世纪前黄河中下游农牧业生产经验
- 《肘后备急方》（晋·葛洪）— 中医治疗学专著

---

## 使用建议

研究**元嘉北伐**及刘宋人物，首选：
1. **《宋书》**（沈约）— 最详，有列传
2. **《南史》**（李延寿）— 补充，跨朝整合
3. **《资治通鉴》**（司马光）— 编年系事，便于理清时序

研究**北魏**，首选：
1. **《魏书》**（魏收）
2. **《北史》**（李延寿）
