# 宋孝武帝刘骏（430—464）

刘宋第四任皇帝，字休龙，[[wiki/entities/刘义隆|宋文帝]]第三子，在位453—464年，年号孝建、大明，庙号世祖。

## 称帝前的积累

刘骏9岁即外放出镇地方，**16岁出镇雍州**，是东晋偏安以来首位出镇襄阳的皇族宗室（《宋书》孝武帝本纪几乎未载，散见各传）：
- **修六门堰**（《宋书·刘秀之传》）："襄阳有六门堰，良田数千顷，堰久决坏，公私废业。世祖遣秀之修复，雍部由是大丰"
- **平缘沔诸蛮**：移一万四千余口于京师，统领[[wiki/entities/沈庆之|沈庆之]]、[[wiki/entities/柳元景|柳元景]]、宗悫等著名将领

20岁守彭城：刘义恭欲弃城，刘骏曰"**必与彭城共存亡**"，彭城守住；又领兵烧北魏辎重杀魏军三千余。

在建康朝中人脉薄弱（母路惠男出身低贱，被高门士族鄙视），宋文帝废太子时讨论储君人选，刘骏直接被排除在外。

## 即位

453年皇太子[[wiki/entities/刘劭|刘劭]]弑父自立。刘骏时任江州刺史，在[[wiki/entities/沈庆之|沈庆之]]辅佐下起兵，联络荆州刺史刘义宣、雍州刺史臧质响应。[[wiki/entities/柳元景|柳元景]]率军新亭大破刘劭，刘骏于453年5月即位，随后攻下建康，斩刘劭及二兄刘濬。

## 核心功臣

辅佐孝武帝起兵、即位及初期稳政的核心圈：
- [[wiki/entities/沈庆之|沈庆之]]（吴兴寒门）——辅立起兵，辅政后期掌军事，遗命"大事与沈庆之决"
- [[wiki/entities/柳元景|柳元景]]（河东北人）——新亭大捷，辅政执掌尚书令
- [[wiki/entities/宗悫|宗悫]]（南阳人）——"愿乘长风破万里浪"，讨刘劭前锋，任左卫将军，封洮阳侯；461年堕马断腿后转任光禄大夫、中护军
- [[wiki/entities/颜竣|颜竣]]——最倚重文臣，458年前；参见幸臣体制节
- [[wiki/entities/颜师伯|颜师伯]]——458年后取代颜竣，宠信冠绝群臣；遗命"尚书中事委颜师伯"

## 即位初稳固措施

五大举措同步推进：
1. **大赦**：蠲今年租税、赦赃污清议；禁奢侈、禁权贵经商、开放江海田池公产
2. **地方人事**：遣十五人巡行风俗，精选各州刺史（申恬治青州、刘秀之治益州、朱修之治雍州，皆有善政）
3. **禁军重组**：[[wiki/entities/沈庆之|沈庆之]]、[[wiki/entities/柳元景|柳元景]]、宗悫分任领军、左卫、右卫将军；省太子东宫武力（率更令、步兵校尉等）；新置殿门及上阖屯兵
4. **封赏功臣**：[[wiki/entities/颜竣|颜竣]]、刘延孙任侍中；[[wiki/entities/戴法兴|戴法兴]]等授要职；沈庆之、柳元景等论功进爵
5. **昭雪刘劭所害者**：徐湛之、江湛、王僧绰等追谥赠官，家属长给禀禄

## 平叛

- **454年**：刘义宣、臧质、鲁爽举兵——沈庆之大破鲁爽，垣护之败刘义宣/臧质，臧质被斩，刘义宣赐死
- **455年**：武昌王刘浑自号楚王，废为庶人赐死
- **459年**：竟陵王刘诞据广陵叛乱，沈庆之平定，城内三千男丁全部斩杀

## 集权改革

- **撤"录尚书事"**：取消顾命辅政制度，一人独揽政权
- **分割大州**：扬州→扬州＋东扬州；荆州大幅分割置湘州、郢州；荆州从三十一郡大州缩为十二郡
- **废南蛮校尉**：将荆州来源重兵营户全部迁至建康，荆州彻底失去"分陕"地位；雍州同步填实为重镇，形成荆雍郢湘四州相互牵制格局，"江陵素畏襄阳人"的局面由此形成
- **设王畿区划**：分扬州六郡及南徐州南琅琊郡共七郡为王畿，作为中央直辖区，地位高于畿外诸州
- **限制地方调兵权**（诏令原文）："自今刺史守宰，动民兴军，皆须手诏施行。唯边隅外警，及奸衅内发，变起仓卒者，不从此例。"
- **抑制诸王**：全面裁损藩王车服器用制度共二十四条
- **省太子武力**：撤销太子率更令、步兵校尉、翊军校尉、旅贲中郎将等东宫武装建制
- 重用[[wiki/entities/沈庆之|沈庆之]]（吴兴寒门）、[[wiki/entities/柳元景|柳元景]]（河东北人），以军功升三公，开创寒门以军功升三公先例

## 幸臣体制："寒人掌机要"

**转折节点（458年）**：[[wiki/entities/颜竣|颜竣]]被外放东扬州、[[wiki/entities/王僧达|王僧达]]被诬杀——士族文臣阶段结束。此后刘骏专任幸臣掌机要：[[wiki/entities/戴法兴|戴法兴]]（最受宠）、巢尚之、戴明宝、徐爰，"事必与议"。史称"主威独运，官置百司，权不外假"。

颜竣是转折前最倚重的文臣，随刘骏平刘劭时病危独入病榻代为决策；王僧达是458年清洗士族的具体案例，临死揭发颜竣怨言致颜竣随后被杀。

**这是[[wiki/entities/宋明帝刘彧|宋明帝]]朝幸臣三人组（[[wiki/entities/阮佃夫|阮佃夫]]、[[wiki/entities/王道隆|王道隆]]、杨运长）的制度前身。**

戴法兴死后民间谣言"戴法兴为真天子，废帝是赝天子"——可见其权重超越皇权感知。

## 殷淑仪与晚年悼亡

[[wiki/entities/殷淑仪|殷淑仪]]（实为[[wiki/entities/刘义宣|刘义宣]]之女，孝武帝堂妹）——孝武帝与刘义宣女儿私通，致使刘义宣454年举兵叛变；义宣败死后，孝武帝将其接入宫，伪称殷氏，封淑仪。

殷淑仪宠冠后宫，生刘子鸾（几乎夺太子位）等五子一女。大明六年（462年）四月殷淑仪去世，孝武帝：
- 棺做成抽屉状随时睹遗容；追赠贵妃谥"宣"
- 精神焕散，不理政事；每晚灵前对饮痛哭
- 招魂仿汉武帝故事，亲作**拟李夫人赋**；谢庄作哀文，"纸与墨价钱大涨"
- 建新安寺（以子刘子鸾封号"新安王"命名）

462—464年孝武帝因此大幅怠政，大明七至八年浙江旱灾期间政事失位，"十分之六户口饿死逃散"。

## 驾崩与顾命

大明八年（464年），孝武帝驾崩，**年仅34岁**。遗诏确立顾命五大臣体制，职权明确分工：

| 人物 | 职权 |
|------|------|
| [[wiki/entities/刘义恭|刘义恭]] | 加中书监，与柳元景"事无巨细，悉关二公" |
| [[wiki/entities/柳元景|柳元景]] | 领尚书令，入住城内，"大事与沈庆之决" |
| [[wiki/entities/沈庆之|沈庆之]] | 掌军事，总统军旅 |
| [[wiki/entities/颜师伯|颜师伯]] | 尚书中事 |
| [[wiki/entities/王玄谟|王玄谟]] | 外监（地方军事监察） |

长子[[wiki/entities/刘子业|刘子业]]（前废帝）继位，旋即杀戴法兴，随后清洗顾命五大臣（沈庆之先被告发，刘义恭、柳元景、颜师伯465年同日被杀）。

## 个人作风

- **乱伦**：《宋书》模糊记载与母路太后有染，《魏书》直言"骏淫乱无度，蒸其母路氏"；儿子[[wiki/entities/刘子业|前废帝刘子业]]指着其画像骂"此渠大好色，不择尊卑"
- **赌博敛财**：命各级官员离任须先赌输财物方准离去，被后代皇帝直接改为强制"献奉"
- **晚年奢华**：拆毁刘裕密室做地基，称刘裕"田舍公"；大明七至八年浙江旱灾，十分之六户口饿死逃散

## 子嗣

共28子：
- 长子[[wiki/entities/刘子业|刘子业]]（前废帝），465年被弑
- 三子刘子勋（晋安王），义嘉之乱名义皇帝，466年被杀
- 夭折10人；被前废帝杀2人；**被[[wiki/entities/宋明帝刘彧|宋明帝刘彧]]杀16人**（义嘉之乱后清洗）

## 对北政策

孝武帝对北采取"稳重保守态度"，北魏文成帝同期推行休养生息，双方总体和平。

### 互市争论

北魏求互市，朝臣意见分化：

**主张互市**：刘义恭、竟陵王刘诞、建平王刘宏、何尚之、何偃

**反对互市**：[[wiki/entities/柳元景|柳元景]]、[[wiki/entities/王玄谟|王玄谟]]、[[wiki/entities/颜竣|颜竣]]、谢庄、檀和之、褚湛之
- **谢庄**："唯利是视"；主张"有余则经略，不足则闭关"
- **颜竣**："夷狄之欲侵暴，正苦力之不足耳"；主张"保境以观其衅"

最终恢复互市。

### 周朗"弃徐齐"论

周朗提出激进主张："历下、泗间，何足独恋"——认为徐齐终将难守，不如主动放弃，与北魏缘淮对峙。此论极端消极，但反映了边境长期残破的现实。

### 边境冲突与青州防线

- **大明元年（457年）**：北魏向无盐，东平太守刘胡出战失利，宋廷遣薛安都、沈法系北讨
- **大明二年（458年）清口之战**：北魏攻击清口，颜师伯指挥宋军击退

青兖频遭侵扰，孝武帝将青州治所迁往历城；开始依赖地方豪族守边——从主动出击到守势的战略退缩。

## 文学与才学

《南史》《资治通鉴》称孝武帝"长于骑射，机警勇决"，文则"**学问博洽，文章华敏**"；《文心雕龙》《诗品》均肯定其诗才出众。鲍照、韩兰英等文人称颂其为"中兴之主"。孝武帝本人创作在南朝文学史上有一定地位，但具体作品在现有文献中记载不多。

年号意义：**孝建**→强调正统继承（正统焦虑）；**大明**→发扬光大正统（确立后的自信）。

## 与蔡兴宗的关系

[[wiki/entities/蔡兴宗|蔡兴宗]]在孝武帝朝以直言著称，孝武帝"惮之不敢侵狎"。孝武帝死后蔡兴宗称之"守道之君"——是刘宋诸帝中少有的正面评价。

## 史料价值

- "寒人掌机要"体制确立，宋明帝幸臣三人组的制度渊源
- 赌博敛财→强制"献奉"的制度演变记录
- 28子被刘彧杀16人的数字权威来源（[[wiki/concepts/义嘉之乱|义嘉之乱]]后清洗规模）
- [[wiki/entities/刘子业|刘子业]]荒暴性格的家族背景（父子皆有乱伦风评）
