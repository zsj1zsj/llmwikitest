# Raw Sources

Immutable source documents. LLM reads from here but never modifies these files.

## Subfolders
- `articles/` — web articles, blog posts
- `papers/` — academic papers, reports
- `clippings/` — Obsidian Web Clipper exports
- `notes/` — your own rough notes, transcripts, voice memos

## How to add a source
1. Drop the file into the appropriate subfolder
2. Tell Claude: `ingest raw/articles/my-article.md`

## Tips
- Use **Obsidian Web Clipper** (browser extension) to clip articles directly as markdown
- After clipping, use "Download attachments" hotkey to save images locally
